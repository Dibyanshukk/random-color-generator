# Random Color Generator 🎨

This is a simple JavaScript project that generates a random color when the button is clicked.

## Features
- Generate random RGB color
- Show color code on screen
- Change box background color

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone this project
2. Open `index.html` in browser
3. Click the "Generate Color" button
